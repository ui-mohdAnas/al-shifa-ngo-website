# Al-Shifa — NGO Website Concept

A single-page concept website design for an NGO, built to showcase the essential features every NGO website should have: donations, volunteer registration, impact metrics, campaigns & events, testimonials, and contact support.

**Live demo:** https://voluble-tapioca-8a7d79.netlify.app/

## Features

- 🔔 Auto-rotating campaign/events notifier bar
- 💝 Itemized donation cards (real, specific costs — not vague pooled funds)
- 🤝 Current volunteer showcase + a working registration form
- 🌟 Live-style impact stats (volunteers, donors, funds raised/spent)
- 🕊️ Dedicated emergency appeal section
- 📅 Campaigns & events list
- 💬 Testimonials carousel
- 📞 Contact section with a working message form

## Tech

Plain HTML/CSS/JS — no framework, no build step, no backend. All visuals are self-contained CSS/SVG gradient tiles (no external image requests). Forms are wired to [FormSubmit.co](https://formsubmit.co) for free, backend-free email delivery.

## Running locally

Just open `index.html` in a browser — no install, no server needed.

## Deployment

Deployed as a static site on Netlify. Any static host works (Netlify, GitHub Pages, Vercel).
